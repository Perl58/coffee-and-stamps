# 📅 Japanese Learning & Motivation Calendar
### 〜 先生と生徒をつなぐ「できた！」の記録帳 〜

**🚧 現在、絶賛開発中（In Development）！ 🚧**

## 🌸 このアプリについて
このアプリは、日本語を学ぶ生徒さんと先生のコミュニケーションを深め、学習のモチベーションを維持するために開発しています。

日々のCLASSの小さな積み重ねを、カレンダー形式で楽しく、可愛く記録します。

## ✨ 開発のきっかけと想い
- **コミュニケーションの活性化**: 生徒さんの頑張りをスタンプやコメントで応援したい。
- **自己肯定感のアップ**: 自分のペースで「できた」を実感できる場所を作りたい。
- **直感的な操作**: MS Accessの堅実なデータ管理と、Webの柔軟で可愛いデザインを融合。

## 🛠 現在の機能（と、これから作ること）
- [x] **カレンダー表示**: 月間の活動をひと目で把握。
- [ ] **グローバル行事表示**: 日本やアメリカの祝日・行事をカレンダーに追加したい！
- [x] **ツールチップ表示**: 🍊や🌟にマウスを乗せると、メモがふわっと出現。
- [ ] **スタンプ投稿フォーム**: その日の気分や学習内容をサクッと入力。
- [ ] **クラウド連携**: どこからでも自分のデータにアクセスできるように。

## 🚀 技術スタック
- **Backend**: Python / Flask
- **Frontend**: HTML5 / CSS3 / JavaScript
- **Concept**: MS Access-inspired data structure
